What is CSS?

Cascading Style Sheets(CSS) is used format the layout of a webpage

With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what
background images or background colors are to be used, different displays for devices and creen sizes, and much more!

Tip:The word cascading means that a styles applied to a parent element will also apply to all children element within the parenrt. So, if you
set the color of the body text to "blue", all headings, paragraphs, and other text elements within the body will also get the same color (unless
you specify something else)!
